# minn-cli

> Quickly find the smallest value in an array `[1,56,9,1925,586,3]` -> `[1]`


## Install

```
$ npm install --global minn-cli
```


## Usage

```
$ min --help

    Usage
        $ min <array>
        echo <array> | min

    Examples
        $ min 99, 2, 1, 4, 5, 8, 29
        1
```
